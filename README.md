# 🤖 AI Knowledge Assistant (RAG)

A production-ready AI Knowledge Assistant built using **Retrieval-Augmented Generation (RAG)**. The application allows users to upload documents, index them into a vector database, ask natural language questions, and receive context-aware answers generated by a local LLM.

---

# 🚀 Features

* Upload PDF, DOCX, CSV, TXT and Markdown documents
* Automatic document chunking
* Embedding generation
* Vector search using ChromaDB
* Retrieval-Augmented Generation (RAG)
* Local LLM inference using Ollama
* Chat history support
* Multi-turn conversation using Chat IDs
* Citation support
* Query grounding score
* Response latency tracking
* Evaluation dashboard
* Export answers as:

  * PDF
  * DOCX
  * Excel
* Dark / Light theme
* Responsive modern UI
* REST API using FastAPI

# ⚙️ Backend Workflow

## 1. Upload Documents

User uploads files.

↓

Parser extracts text.

↓

Text is split into chunks.

↓

Embeddings are generated.

↓

Chunks are stored in ChromaDB.

---

## 2. Ask Question

User submits a question.

↓

Retriever searches the most relevant chunks.

↓

Prompt Builder creates the final prompt.

↓

Ollama generates the answer.

↓

History is saved.

↓

Evaluation metrics are calculated.

↓

Answer with citations is returned.

# 🖥 Installation

## Clone Repository

```bash
git clone https://github.com/2662Debashis008/ai-knowledge-assistant.git

cd ai-knowledge-assistant
```

---

## Backend Setup

```bash
cd backend

python -m venv .venv

source .venv/bin/activate
```

Windows

```powershell
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
uvicorn main:app --reload
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

# 🌐 API Endpoints

| Method | Endpoint      | Description             |
| ------ | ------------- | ----------------------- |
| POST   | /upload       | Upload documents        |
| POST   | /ask          | Ask questions           |
| POST   | /new-chat     | Create chat             |
| GET    | /history      | Chat history            |
| DELETE | /history/{id} | Delete chat             |
| GET    | /documents    | List uploaded documents |
| GET    | /metrics      | Metrics                 |
| GET    | /evaluation   | Evaluation              |
| POST   | /export/pdf   | Export PDF              |
| POST   | /export/docx  | Export DOCX             |
| POST   | /export/excel | Export Excel            |
